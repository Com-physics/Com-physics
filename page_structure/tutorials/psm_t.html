Calculating Pauli Spin Matrix with our Commutator function in GNU Octave

This tutorial demonstrates how to compute the commutator of Pauli spin matrices using an GNU Octave function.

What is a Commutator?

In Quantum Mechanics, the commutator of two operators A, B is the operator:

$$ C = [A, B] = AB - BA $$

Pauli Spin Matrices

The Pauli matrices are complex arrays that arise in the treatment of spin in quantum mechanics.

  • \[ S_x = \frac{\hbar}{2} \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} \]
  • \[ S_y = \frac{\hbar}{2} \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix} \]
  • \[ S_z = \frac{\hbar}{2} \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix} \]
  • but we will work with their unitless equivalents
  • \[ \sigma_x = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} \]
  • \[ \sigma_y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix} \]
  • \[ \sigma_z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix} \]

Step 1: Define the Commutator Function

Create a file named commutator.m with the following content:

function C = commutator(A, B)
  AB = A * B;
  BA = B * A;
  C = AB - BA;
end

Step 2: Compute Commutators in GNU Octave

Open GNU Octave and define the Pauli spin matrices:

sigma_x = [0 1; 1 0];
sigma_y = [0 -1i; 1i 0];
sigma_z = [1 0; 0 -1];

Then compute a commutator, for example \([\sigma_x, \sigma_y]\):

commutator(sigma_x, sigma_y)

Expected Results

    >> commutator(sigma_x, sigma_y)
ans =

   0 + 2i   0 + 0i
   0 + 0i   0 - 2i
  • \(which \ is,\)
  • \([\sigma_x, \sigma_y] = 2i \sigma_z\)
  • \(Similarly,\)
  • \([\sigma_y, \sigma_z] = 2i \sigma_x\)
  • \([\sigma_z, \sigma_x] = 2i \sigma_y\)
